# IEEE Template

Drop `ieee_template.docx` here (the Golden IEEE Word template used by the rendering engine).

The source file is at `reference/conference-template-a4.docx` in the repo root.
Copy it here and rename it `ieee_template.docx` before running Milestone 2.

Named styles the renderer expects (do not rename them in the template):
- `IEEE Title`
- `IEEE Author`
- `IEEE Abstract`
- `IEEE Keywords`
- `IEEE Section Heading` (level 1)
- `IEEE Subsection Heading` (level 2)
- `IEEE Subsubsection Heading` (level 3)
- `IEEE Body Text`
- `IEEE Caption`

These will be verified and documented fully in Milestone 2 after inspecting the actual template.
